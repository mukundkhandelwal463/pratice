# Sample HTML/CSS/JS + Node Project

A simple sample project using Node.js as a static server and API backend.

## Setup

1. Open a terminal in this folder.
2. Run `npm install`.
3. Run `npm start`.

## Usage

- Open `http://localhost:3000` in your browser.
- Click the button to fetch a message from the Node server.

## Files

- `server.js` — Node server using Express.
- `public/index.html` — Main HTML page.
- `public/style.css` — Styles.
- `public/app.js` — Browser JavaScript fetching the API.
